Form – Sending Information to Another Page

This example shows how to send information from one page (form.html) to another page (form-results.html) using an HTML form.

The form below includes:

  • a text box
  • radio buttons
  • a list selector
  • a choice box (checkbox)
  • a hidden parameter
  • and an email field for sending results

Example Form Code

<form action="form-results.html" method="get">
    <!-- Text box -->
    <label for="name">Your Name:</label>
    <input type="text" id="name" name="name">

    <!-- Email field -->
    <label for="email">Your Email (to receive results):</label>
    <input type="email" id="email" name="email">

    <!-- Radio buttons -->
    <p>Favorite drink:</p>
    <input type="radio" id="coffee" name="drink" value="Coffee">
    <label for="coffee">Coffee</label>
    <input type="radio" id="tea" name="drink" value="Tea">
    <label for="tea">Tea</label>
    <input type="radio" id="refresher" name="drink" value="Refresher">
    <label for="refresher">Refresher</label>

    <!-- List selector -->
    <label for="course">Course you are working on:</label>
    <select id="course" name="course">
        <option value="database">Database</option>
        <option value="html">HTML</option>
        <option value="oldtestament">Old Testament</option>
    </select>

    <!-- Choice box (checkbox) -->
    <label>
        <input type="checkbox" name="newsletter" value="yes">
        I would like to receive updates.
    </label>

    <!-- Text area (text box) -->
    <label for="comments">Comments:</label>
    <textarea id="comments" name="comments"></textarea>

    <!-- Hidden parameter -->
    <input type="hidden" name="hiddenValue" value="csc210-demo">

    <input type="submit" value="Send Form">
</form>

Working Example

The form even sends along the email address you entered, so in a real website a server could email your answers back to you.
