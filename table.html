How to Make a Table in HTML

In HTML, tables are created using the <table> tag. Each row is inside <tr>, headers go in <th>, and regular cells go in <td>.

Basic Example

This is an example of the code for a simple table:

<table>
    <tr>
        <th>Class</th>
        <th>Day</th>
        <th>Time</th>
    </tr>
    <tr>
        <td>Database Class</td>
        <td>Monday</td>
        <td>6:00 PM</td>
    </tr>
    <tr>
        <td>Old Testament</td>
        <td>Wednesday</td>
        <td>3:00 PM</td>
    </tr>
</table>

What the Table Looks Like

Class Day Time
Database Class Monday 6:00 PM
Old Testament Wednesday 3:00 PM
