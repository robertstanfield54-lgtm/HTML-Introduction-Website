How to Make a Bulleted List in HTML

In HTML, a bulleted list is made with the <ul> tag (unordered list). Each item in the list goes inside an <li> tag (list item).

Basic Example

This is the code for a simple bulleted list:

<ul>
    <li>Make coffee</li>
    <li>Work on HTML homework</li>
    <li>Study for Old Testament class</li>
</ul>

What the List Looks Like

  • Make coffee
  • Work on HTML homework
  • Study for Old Testament class
