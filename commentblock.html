How to Make a Comment Block in HTML

← Back to Home

What is a Comment?

Comments let you leave notes in your HTML that the browser ignores. They are useful for explaining your code to yourself or others.

Basic Comment Syntax

A simple comment looks like this:

<!-- This is a single-line comment -->

Comment Block Example

You can also write a multi-line comment block:

<!-- 
    This is a comment block.
    You can write multiple lines of notes here.
    The browser will ignore all of this.
-->

Step-by-Step Instructions

  1. Open your HTML file in your editor.
  2. Decide where you want to add the comment.
  3. Type <!-- to start the comment.
  4. Write your note or explanation.
  5. Type --> to end the comment.
