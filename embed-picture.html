How to Embed a Picture in a Website

To add a picture to a web page, you use the <img> tag. The most important part is the src attribute, which tells the browser where the image file is located.

Basic Example

This is an example of the code to show a picture called landscape.jpg:

<img src="wintertree.jpg" alt="Landscape photo">
