How to Show Three Different Fonts

You can change the font type, color, and size of text using CSS. One way to do this is to create classes in the <style> section and then apply them to different paragraphs.

Example CSS and HTML

Here is an example of the code:

<style>
    .example1 {
        font-family: "Times New Roman", serif;
        font-size: 20px;
        color: darkred;
    }

    .example2 {
        font-family: "Courier New", monospace;
        font-size: 18px;
        color: darkblue;
    }

    .example3 {
        font-family: Verdana, sans-serif;
        font-size: 22px;
        color: darkgreen;
    }
</style>

<p class="example1">This is Times New Roman, dark red, 20px.</p>
<p class="example2">This is Courier New, dark blue, 18px.</p>
<p class="example3">This is Verdana, dark green, 22px.</p>
