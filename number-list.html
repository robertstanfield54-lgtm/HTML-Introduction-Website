How to Make a Numbered List in HTML

In HTML, a numbered list is made with the <ol> tag (ordered list). Each item in the list goes inside an <li> tag (list item).

Basic Example

This is the code for a simple numbered list:

<ol>
    <li>Wake up</li>
    <li>Go to work at the coffee shop</li>
    <li>Come home and work on HTML homework</li>
</ol>

What the List Looks Like

  1. Wake up
  2. Go to work at the coffee shop
  3. Come home and work on HTML homework
