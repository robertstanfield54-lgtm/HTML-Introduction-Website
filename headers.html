How to Make Headers of Three Different Sizes

HTML has built-in heading tags that range from <h1> (largest) to <h6> (smallest). Here we will use three of them: <h1>, <h2>, and <h3>.

Example Code

<h1>This is an H1 heading</h1>
<h2>This is an H2 heading</h2>
<h3>This is an H3 heading</h3>
